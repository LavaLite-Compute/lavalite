# NAME

**bparams** - displays information about configurable system parameters
in lsb.params

# SYNOPSIS

**bparams** \[**-l**\]

**bparams** \[**-h** \| **-V**\]

# DESCRIPTION

Displays information about configurable system parameters in lsb.params.

By default, displays only the interesting parameters.

# OPTIONS

**-l**

:   

    Long format. Displays detailed information about all the
    configurable parameters in lsb.params.
